#### ArrayLike

```typescript
// lib.es5.d.ts
interface ArrayLike<T> {
    readonly length: number;
  	// 可索引属性
    readonly [n: number]: T;
}
```

> 这个是在看到React源码时，发现很多是使用原型的call方法去调用，而不是使用对象调用。如 Array.prototype.slice.call 的方式调用，因为这样有很好的兼容性，参数可以是ArrayLike类型。如函数的参数arguments
